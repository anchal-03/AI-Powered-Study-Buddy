# AI-Powered-Study-Buddy
This is my first project

## 📌 Overview

An AI-based learning assistant that helps students understand concepts, summarize notes, and generate quizzes and flashcards.

## 🚀 Features

* Simple topic explanations
* Note summarization
* Quiz generation
* Flashcard creation
* Personalized learning support

## 🛠️ Technologies Used

* Python
* AI & NLP
* Streamlit/Gradio
* Hugging Face / OpenAI API

## ⚙️ How It Works

1. User enters a topic or notes.
2. AI processes the input.
3. The system provides explanations, summaries, quizzes, and flashcards.

## 🔮 Future Scope

* Voice support
* Multilingual support
* Progress tracking
